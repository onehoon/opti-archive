# OptiScaler Binary Assets 📦

This repository serves as a dedicated distribution storage for **OptiScaler** binary files, used by the [OptiScalerInstaller](https://github.com/onehoon/OptiScalerInstaller) project.

## 🚀 Purpose
To provide a clean and reliable source for pre-built OptiScaler assets without cluttering the main installer repository. This separation ensures that users always have access to the specific versions required for game optimization.
